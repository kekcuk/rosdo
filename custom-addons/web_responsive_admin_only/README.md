# Web Responsive — Admin Only

## Описание
Этот модуль скрывает элементы пользовательского меню "Apps" и "Preferences/Settings" для всех пользователей, кроме администраторов.

## Функциональность
- Скрывает меню "Apps" для не-администраторов
- Скрывает меню "Preferences/Settings" для не-администраторов
- Скрывает элементы настройки темы для не-администраторов

## Требования
- Модуль `web_responsive`

## Установка
1. Установите модуль `web_responsive_admin_only`
2. Убедитесь, что пользователи, которым нужно предоставить доступ к этим элементам меню, принадлежат к группе "Settings" (base.group_system)

## Примечание
Этот модуль создан для решения проблемы, при которой все пользователи могут видеть меню "Apps & Preferences" в Odoo 18 с включенным модулем Web Responsive.
