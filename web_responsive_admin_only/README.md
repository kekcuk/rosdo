# Web Responsive Admin Only

## Описание
Модуль скрывает элементы интерфейса, предоставляемые модулем web_responsive, от обычных пользователей. Только администраторы (пользователи с правами system) могут видеть и использовать:

- Каплю в трее для смены темы
- Переключатель типа поиска
- Другие элементы мобильного интерфейса

## Особенности
- Использует JavaScript патчинг для определения прав пользователя
- Скрывает элементы интерфейса как в шаблонах, так и через DOM манипуляции
- Совместим с Odoo 18
- Не требует наследования от специфических шаблонов
- Использует MutationObserver для динамического скрытия элементов
- Работает с различными версиями модуля web_responsive

## Установка
1. Установите модуль web_responsive (если еще не установлен)
2. Установите данный модуль
3. Обновите страницу браузера (очистите кэш)
4. Проверьте, что обычные пользователи не видят каплю в трее и не могут изменять тему

## Технические детали
Модуль работает на двух уровнях:
1. JavaScript-патч компонента UserMenu для проверки прав пользователя
2. DOM-манипуляции для скрытия элементов интерфейса по CSS-селекторам
3. MutationObserver для отслеживания динамически создаваемых элементов

## Совместимость
- Odoo 18.0
- Все версии модуля web_responsive

## Устранение неполадок
Если элементы интерфейса все еще видны обычными пользователями:
1. Проверьте, что модуль установлен и активен
2. Очистите кэш браузера
3. Перезагрузите страницу
4. Убедитесь, что пользователь не состоит в группе "Администрирование / Настройка системы"
